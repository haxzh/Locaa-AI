💡 API ENDPOINTS REFERENCE
───────────────────────────────────────────────────────────────────────

IMAGE GENERATION:
  curl -X POST http://localhost:5000/api/ai/generate-image \
    -H "Authorization: Bearer $TOKEN" \
    -H "Content-Type: application/json" \
    -d '{
      "prompt": "Your prompt here",
      "size": "1024x1024",
      "quality": "standard"
    }'

VIDEO GENERATION:
  curl -X POST http://localhost:5000/api/ai/generate-video \
    -H "Authorization: Bearer $TOKEN" \
    -H "Content-Type: application/json" \
    -d '{
      "prompt": "Your prompt here",
      "duration": 4
    }'

GET HISTORY:
  curl -X GET "http://localhost:5000/api/ai/history?type=image&limit=20" \
    -H "Authorization: Bearer $TOKEN"

❓ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────

"OpenAI API key not configured"
→ Check .env has OPENAI_API_KEY=sk_...
→ Restart Flask

"Rate limit exceeded (OpenAI)"
→ Wait 1 minute, free tier limited to 3 req/min

"Video generation timed out"
→ Normal for videos (takes 30-60s+)
→ Check REPLICATE_API_TOKEN is valid

"Failed to fetch history"
→ Check JWT token valid
→ Verify user authenticated
→ Check browser console for errors

"Image/Video URL returns 404"
→ External URLs expire after 1 hour
→ Save files to S3/Blob for persistence
